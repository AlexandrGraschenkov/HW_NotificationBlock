HW_NotificationBlock
====================

Ваща цель реализовать паттерн Наблюдатель(Observer) на `ObjC`. Ничем вспомогательным пользоватся нельзя. Пользуйтесь тестами для проверки работоспособность (<kbd>⌘</kbd>+<kbd>U</kbd>).
